# 📊 Employee Attrition Risk Dashboard

## 📌 Project Overview
This project focuses on analyzing employee data to identify individuals who are at risk of leaving the organization. It uses survey-based inputs and key HR metrics to calculate an **attrition risk score**, categorize employees, and provide actionable insights through a dashboard.

The goal is to help organizations take **proactive decisions** to improve employee retention and engagement.

---

## 🚀 Features
- 📥 Data input through structured datasets (Excel/CSV)
- 🧮 Attrition risk score calculation
- 🚦 Employee classification:
  - Low Risk
  - Medium Risk
  - High Risk

## ⚙️ How It Works
1. Collect employee data (survey responses, working hours, satisfaction, etc.)
2. Process and clean the data
3. Calculate risk scores based on defined parameters
4. Categorize employees into risk levels
5. Generate alerts for high-risk employees
6. Visualize results in a dashboard

---

## 📊 Key Parameters Used
- Employee Satisfaction  
- Work Hours  
- Manager Support  
- Career Growth Opportunities  
- Leave Intention
